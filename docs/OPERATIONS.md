# 運用マニュアル

## 代表向け: チーム運営の始め方

### 1. アカウント作成
1. mound にアクセス
2. 「LINE で始める」→ LINE アカウントでログイン
3. チーム名と活動エリアを入力 → チーム作成完了

### 2. メンバー登録
1. サイドメニュー → 「メンバー」
2. 「メンバーを追加」ボタン
3. 名前・ポジション・背番号を入力
4. 設定ページから LINE 招待リンクを生成してメンバーに共有

### 3. グラウンド登録
1. サイドメニュー → 「グラウンド」
2. 「グラウンドを追加」
3. 名前・市区町村・料金・設備情報を入力

### 4. 対戦相手登録
1. サイドメニュー → 「対戦相手」
2. 「対戦相手を追加」
3. チーム名・エリア・連絡先を入力

### 5. 助っ人登録
1. サイドメニュー → 「助っ人」
2. 「助っ人を追加」
3. 名前・連絡先を入力

## 試合運営フロー

### 試合の作成
1. 「活動を作成」→ 種別（試合/練習/イベント）を選択
2. 日時・場所・最低人数を設定
3. 作成すると自動で出欠収集が開始

### 出欠の収集
- **LINE**: LIFF 経由で自動通知（LINE 連携済みメンバー）
- **Web**: 試合詳細 → 「出欠リンクを生成」→ メンバーにリンクを共有
- **代理入力**: 出欠状況テーブルから直接変更可能

### 対戦交渉
1. 試合詳細 → 「対戦交渉を管理」
2. 「交渉を作成」→ 対戦相手選択 → 候補日追加
3. ステータス管理: 下書き → 送信済み → 返信あり → 成立/不成立

### 助っ人打診
1. 試合詳細 → 「助っ人を管理」
2. 「助っ人を打診」→ 複数選択 → メッセージ入力
3. 承諾/辞退を追跡

### 試合確定
- 出欠状況が最低人数を満たし、対戦相手が確定したら
- ステータスを「確定」に変更

### 試合当日〜結果入力
1. 「出席を記録」→ 実際の出席/ドタキャンを記録
2. 「試合結果を入力」→ スコア入力
3. 「打席結果を入力」→ 個人成績入力（任意）
4. 「投球成績を入力」→ ピッチャー成績入力（任意）

### 精算
1. 「支出・精算を管理」→ 経費登録
2. 自動で一人あたりの金額を計算
3. PayPay リンク生成 → メンバーに送信

## Cron ジョブ（自動処理）

| ジョブ | 間隔 | 内容 |
|--------|------|------|
| orchestrate | 30分 | 次のアクション提案生成 |
| reminders | 毎日9時/18時 | 出欠リマインダー送信 |
| deadlines | 毎時 | 締切処理（未回答→不参加） |
| grounds | 毎日6時 | グラウンド空き状況チェック |

## データバックアップ

Supabase Dashboard → Settings → Database → Backups でバックアップを管理。
自動バックアップは Supabase Pro プラン以上で利用可能。
